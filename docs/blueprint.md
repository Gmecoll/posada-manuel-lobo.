# **App Name**: Hotel Hub

## Core Features:

- Check-in Management: Allows hotel staff to manage guest check-ins, record arrival times, and assign rooms.
- QR Code Generation: Generates unique QR codes for each guest, providing secure and keyless room access.
- Room Access Control: Enables guests to unlock their assigned rooms by scanning the QR code using their smartphone.
- Booking management: Access, modify, and create bookings.

## Style Guidelines:

- Primary color: Deep turquoise (#45B8AC), evoking a sense of calm and sophistication, like tranquil waters.
- Background color: Light beige (#F5F5DC), offering a warm and inviting backdrop reminiscent of sandy beaches.
- Accent color: Coral (#FF7F50), used sparingly to highlight important interactive elements and call attention to key actions.
- Font Pairing: 'Playfair' (serif) for headlines, lending a touch of elegance, and 'PT Sans' (sans-serif) for body text, ensuring readability.
- Use clean, minimalist icons to represent hotel services and room amenities, ensuring easy recognition and visual clarity.
- Employ a grid-based layout to create a visually organized and easy-to-navigate interface.
- Implement subtle transitions and animations to enhance user engagement and provide feedback for interactions, such as QR code scanning.
